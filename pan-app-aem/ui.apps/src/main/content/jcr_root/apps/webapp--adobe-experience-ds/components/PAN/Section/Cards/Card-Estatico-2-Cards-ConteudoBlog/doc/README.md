Dialog (modal de edição)

1 **Palavra Destaque**  

    - Campo do tipo texto simples  

    Configuração na Dialog (modal de edição):  
    Aba: Textos  
    Campo: Palavra Destaque  
&nbsp; 

2 **Título**

    - Campo do tipo rich text  

    Configuração na Dialog (modal de edição):  
    Aba: Textos  
    Campo: Título  
&nbsp; 

3 **Cards**

    Possui 2 abas, sendo uma para cada card no componente:
    
    3.1 - Necessário preencher a palavra destaque de cada card
    3.2 - Necessário preencher o título de cada card
    3.3 - Necessário inserir a imagem de cada card (552x374), bem como seu texto alternativo
    3.4 - Necessário configurar o botão de cada card (dialog própria, preencher por último)
    
    Configuração na Dialog (modal de edição):  
    Aba: Cards > Card X (Onde X representa o índice/número daquele card)
&nbsp; 


### Definições de SEO e Acessibilidade
Aba SEO:

- ID da seção (para uso quando algum botão precisar de scroll)
- Definições das tags HTML referente aos campos: Palavra destaque e Titulo

Acessibilidade:

- Texto alternativo para as imagens de cada card
