Aba: Imagem

2- Icone

- Resolução base: 32x32 (área útil)
- Necessário definir atributo ALT
- Aceita apenas .svg
Configuração na Dialog (modal de edição):
Aba: Card

3 - Titulo

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Card
Campo: Título

4 - Texto

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Card
Campo: Texto

5 - Link

- Campo de texto simples
- Definição de comportamento (link ou scroll para seção escolhida)
Configuração na Dialog (modal de edição):
Aba: Card

6 - Target Link

- Campo do tipo seletc
- Definição de comportamento (direcionar em uma nova página ou na mesma)
Configuração na Dialog (modal de edição):
Aba: Card
