**Variantes do componente**
Esse componente contem variantes de cores de fundo e de como será o estilo do card, são elas:

Background:
- Azul
- Branco
- Cinza
- Preto

Estilo do Card:
- Fundo sólido (imagem de exemplo: componente-card-carrossel-cardCentral--solido)
- Fundo transparente com borda (imagem de exemplo: componente-card-carrossel-cardCentral--transparente)

Aba: Cards Carrossel
Campo: Insira os Cards (um para cada)
