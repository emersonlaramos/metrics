Dialog (modal de edição)

[IMAGEM DIALOG]


1 - Palavra Destaque

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Palavra Destaque

2 - Titulo

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Titulo

3 - Image Card

- Campo de imagem tipo WEBP (312x304)
Configuração na Dialog (modal de edição):
Aba: Cards Cartões
Sub-Aba: Imagem
Campo: Imagem

4 - Nome Card / Subtitulo

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Cards Cartões
Sub-Aba: Textos
Campo: Nome Card / Sub-Texto Card

5 - Topicos Card

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Cards Cartões
Sub-Aba: Textos
Campo: Tópico 1 / Tópico 2 / Tópico 3

6 - Button

- Componente padrão Mahoe-Button (Tem dialog própria, clique sobre ele para editar)

7 - Link

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Cards Cartões
Sub-Aba: Link
Campo: CTA Link ou ID da seção para scroll
