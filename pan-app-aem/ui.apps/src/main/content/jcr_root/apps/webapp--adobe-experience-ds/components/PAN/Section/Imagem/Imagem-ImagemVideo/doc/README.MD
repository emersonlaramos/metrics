# Imagem-ImagemVideo
## Instruções de uso do componente
Para visualição dessas variantes veja no [FIGMA](https://www.figma.com/file/CvO1OUqdbSXp6UU1Hn2Bi9/DS-Mahoe---Mkt-Lib?node-id=2%3A2027&t=syt2fC2rysd0IJbj-1)

Dialog (modal de edição)
1 - Titulo
- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Título

2 - Subitulo
- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: SubTitulo

2 - Disclaimer
- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Disclaimer

2 - Botão
- Definição de Data-track (SEO)
- Definição de link 
- Definição de title 
Configuração na Dialog (modal de edição):
Aba: Botão (Play)

3 - Imagem
- Resolução base desktop:  1440 x 582 (área útil)
- Resolução base mobile:  375 x 628 (área útil)
- Ocupar 12 colunas dentro do container
- Necessário definir atributo ALT limitado a 50 caracteres
- Aceita apenas .WEBP
- Necessário subir Desktop e Mobile
Configuração na Dialog (modal de edição):
Aba: Imagem (Imagem de fundo Mobile e Imagem de fundo Desktop)

3 - Imagem
- Resolução ícone botão PLAY: 80x80 (área útil)
- Necessário definir atributo ALT limitado a 50 caracteres
- Aceita apenas .WEBP
Configuração na Dialog (modal de edição):
Aba: Imagem (Ícone play vídeo)

### Definições de SEO
Aba SEO:
- ID da seção (para uso quando algum botão precisar de scroll)
- Definições da tag HTML referente ao campo: Titulo
- As configurações do botão se encontram na aba do mesmo
