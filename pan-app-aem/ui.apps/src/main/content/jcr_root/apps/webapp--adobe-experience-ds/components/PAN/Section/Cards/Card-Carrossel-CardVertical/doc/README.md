Dialog (modal de edição)

[IMAGEM DIALOG]


1 - Palavra Destaque

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Palavra Destaque

2 - Titulo

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Textos
Campo: Titulo

3 - Cards

- Campo Icone do tipo svg
- Campo Título do tipo texto simples
- Campo Texto do tipo texto simples
- Inclusão de imagem de fundo (será um card normal caso não adicione imagem)
- Definição de Data-track (SEO)
- Definição de comportamento (link ou scroll para seção escolhida)
- Pode-se adicionar "n" cards
Configuração na Dialog (modal de edição):
Aba: Cards Carrossel
Campo: Cards (um para cada)
