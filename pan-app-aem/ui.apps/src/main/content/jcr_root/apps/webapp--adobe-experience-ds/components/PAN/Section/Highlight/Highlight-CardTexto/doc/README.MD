## Instruções de uso do componente


**Variantes do componente**
Esse componente contem variantes de cores, existência ou não de ícone no CTA e posicionamento de ícone no CTA, são elas:

Background Superior e Inferior:
- Branco (em ambos)
- Azul (em ambos)
- Preto (em ambos)
- Cinza (em ambos)
- Superior: Cinza /  Inferior: Branco
- Superior: Preto /  Inferior: Branco
- Superior: Azul /  Inferior: Branco
- Superior: Branco /  Inferior: Cinza
- Superior: Branco /  Inferior: Preto
- Superior: Branco /  Inferior: Azul

Background do Card:
- Preto 
- Cinza 
- Azul 
- Azul Dark 

Ícone no CTA:
- Pode ou não ter

Posição de ícone no CTA:
- Justificado 
- Direita 
- Esquerda
