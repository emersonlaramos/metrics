Dialog (modal de edição)

[IMAGEM DIALOG]


1 - Cards

Exibição de 2 cards (obrigatoriamente), cada um deles contém:

1.1 - Titulo

- Campo do tipo texto simples
Configuração na Dialog (modal de edição):
Aba: Cards > Card X (onde X representa o número do card)
Campo: Título

1.2 - Parágrafo

- Campo do tipo texto simples
Obs.: O mesmo pode ser ocultado ao marcar a opção "Ocultar Parágrafo" localizada acima do campo.
Configuração na Dialog (modal de edição):
Aba: Cards > Card X (onde X representa o número do card)
Campo: Parágrafo

1.3 - Imagem
- Imagem em modelo de background seguindo o DS 
- Não tem atributo ALT para imagens em Background no código
- Aceita apenas .WEBP
Configuração na Dialog (modal de edição):
Aba: Cards > Card X (onde X representa o número do card)
Campo: Imagem

1.4 - Button

- Componente padrão Mahoe-Button, possui dialog própria.
